# COVID-19 Vaccination EDA

## Problem Statement
Analyzing global COVID-19 vaccination progress to uncover trends across countries, time periods, and vaccine types.

## Dataset
- Source: Kaggle — COVID-19 World Vaccination Progress
- Rows: 86,213 | Columns: 15

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab

## Key Insights
1. China and India led globally in total vaccinations
2. Global daily vaccinations peaked in mid-2021, then declined
3. Small nations like Gibraltar and Pitcairn achieved the highest % fully vaccinated
4. India surpassed USA in daily doses from mid-2021 onward
5. Oxford/AstraZeneca was the most widely used vaccine across countries
